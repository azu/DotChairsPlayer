# DotChairsPlayer

[ドットインストール](http://dotinstall.com/ "ドットインストール - 3分動画でマスターする初心者向けプログラミング学習サイト") のレッスン一覧から
動画をまとめてダウンロードしてオフラインで見るiPadアプリ

## Usage

![gif](http://gyazo.com/fd02bba6529711b6a8638674f5716345.gif)

## Installation

``` sh
git clone
pod install
open DotChairsPlayer.workspace
# Build iPad.app
```
